# Personal Portfolio

A modern, responsive portfolio website showcasing projects, blog posts, and technical writing.

## 🌐 Live Site

Visit the live portfolio at the deployed URL.

## ✨ Features

- **Responsive Design** - Works on desktop, tablet, and mobile
- **Blog Section** - Technical blog posts on AI agents, Git, Vim, and more
- **Japanese Support** - Bilingual content (English & Japanese)
- **Data Visualizations** - Interactive charts and visualizations
- **Security Headers** - Configured with proper security headers via `_headers`

## 📁 Structure

```
├── index.html           # Main portfolio page
├── index_ja.html        # Japanese version
├── blog/                # Blog post pages
│   └── *.html          # Individual blog posts
├── css/                 # Stylesheets
├── js/                  # JavaScript files
├── img/                 # Images and assets
├── visualizations/      # Data visualization pages
└── vendor/              # Third-party libraries
```

## 🚀 Development

This is a static HTML/CSS/JS site. To develop locally:

```bash
# Using Python's built-in server
python3 -m http.server 8000

# Or using Node's http-server
npx http-server
```

Then open `http://localhost:8000` in your browser.
